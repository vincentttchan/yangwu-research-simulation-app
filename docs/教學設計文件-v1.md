> 遊戲路徑：`探索版/` · Vite dev → `npm run dev`（localhost:5173）

## 一、設計理念與理論依據

本遊戲是**歷史思維模擬工具**，核心不是記憶史實，而是讓學生親歷因果、做抉擇、事後反思。以下七個教育理論直接對應遊戲已有的機制：

| 理論 | 遊戲機制 |
|---|---|
| **Kolb 體驗學習圈**（具體→反思→抽象→再行動） | 玩→結算史評→概念化→重玩換路線 |
| **Vygotsky 鷹架／ZPD** | 新手引導 Coach、年份提示、待解鎖事件 |
| **Seixas 歷史思維六概念** | 查證據(evidence)、條件選項(contingency)、三軸(continuity/change)、七幕(significance)、人物後日談(perspective)、結局代價(ethical dimension) |
| **Bloom 認知層級** | 求知考驗由記憶→分析；結算史評推向評鑑層 |
| **問題導向學習 PBL** | 懸案「海防 vs 塞防」「主和緩兵」為 ill-structured problem，無完美解 |
| **Mayer 多媒體學習** | 水墨地圖＋銅版畫＋音景三碼並進 |
| **自我決定論 SDT** | 自主（選路線/城市）、勝任（功業錄成就）、連結（人物冊） |

## 二、教學目標對應 DSE 課綱

**課綱單元**：洋務運動（約 1861–1895）

| DSE 評核要求 | 本遊戲覆蓋 |
|---|---|
| 洋務運動背景（內憂外患） | 序章圓明園劫火＋咸豐元年開場 |
| 洋務內容（求強/求富） | 九城事件＋見識三軸（器物/制度/思想） |
| 局限與失敗原因 | 條件選項後果＋七幕結局代價 |
| 對比明治維新 | 城市「日本」路線＋東亞變局事件 |
| 甲午戰爭與《馬關條約》 | 黃海一役＋馬關條約結局 |
| 史料運用（SBQ） | 「手卷·證據簿」查證環節 |
| 史論寫作（essay） | 結算個人化史評可作論點素材 |

## 三、三種課堂模式（完整流程）

---

### 模式 A：全班課堂引導（1–2 節課）

**適用**：引起動機、新課導入、概念鞏固

**教師角色**：帶玩者、提問者、debrief 主持人

#### 流程（90 分鐘版）

| 時段 | 活動 | 說明 |
|---|---|---|
| 0–10 min | **序章放映** | 投影序章（圓明園劫火＋引文）。問：「為何一把大火，開啟三十年自強？」 |
| 10–15 min | **背景簡介** | 教師補充《北京條約》前後史況，為遊戲期間打下史實底子 |
| 15–50 min | **拼圖分組玩** | 4 組各玩一路線（李鴻章器物／奕訢制度／容閎思想／自由探索），各記見識三軸得分與選擇結果 |
| 50–65 min | **組際分享** | 各組報告：玩了哪座城、做了什麼抉擇、見識如何增長 |
| 65–80 min | **Debrief（最重要）** | 教師引導：「各組看見不同面向的洋務——為何辦洋務的人要分頭做？他們有沒有統一的領導？」→ 引出洋務缺乏整體規劃的結構性局限 |
| 80–90 min | **反事實澄清** | 強調：遊戲允許玩家「改變」，但甲午慘敗是結構性困局，不是個人選擇能扭轉的。以七幕結局為證。 |

**評估**：出口票（exit ticket）— 寫一句「洋務最根本的局限是什麼」。

---

### 模式 B：自學／翻轉課堂（課前回家玩 + 課堂討論）

**適用**：翻轉課堂、補底、差異化教學

**教師角色**：設計任務、課堂討論主持

#### 課前任務（學生在家完成）

發出以下學習指引：

> **《自強三十年》自學任務**
>
> 1. 完成至少 **一條路線**（到 1895 年結局）
> 2. 截圖你的**結算頁**（見識三軸、歷史時刻、史評）
> 3. 填寫下方學習單（見附件）後帶返課室

#### 課前學習單（Teacher 可印發）

```
姓名：____________  路線：____________  結局：____________

1. 你在遊戲中做了哪一個最難的抉擇？為什麼難？
   ___________________________________________________

2. 你的史評提到洋務的哪個成敗原因？你同意嗎？為什麼？
   ___________________________________________________

3. 遊戲裡有沒有什麼令你意外的歷史事實？
   ___________________________________________________

4. 如果你是李鴻章，你最想改變什麼決定？
   ___________________________________________________
```

#### 課堂活動（45 分鐘）

1. **配對分享**（10 min）：兩人比較結算截圖，找出「相同與不同」
2. **全班討論**（25 min）：
   - 哪個城市／事件讓你學到最多？
   - 第 4 題：反事實討論 → 教師收口「個人選擇 vs 結構制約」
3. **連接 DSE**（10 min）：教師展示 DSE 真題，引導學生用遊戲經歷組織論點

---

### 模式 C：DSE 備試工作坊（高階，1 節課）

**適用**：考前兩至四個月，鞏固論證技巧

**教師角色**：論證教練

#### 活動設計

**步驟一（15 min）：論點提取**

學生打開遊戲結算頁（或截圖），從史評＋歷史時刻中提取可用論點，填入下表：

| 論點 | 支持史例（遊戲見） | DSE 答題可怎樣用 |
|---|---|---|
| 洋務以器物為主，忽略制度 | 見識「思想」一軸難以提升 | 分析洋務失敗原因 |
| 缺乏中央統籌 | 分組路線各自發展，無法互補 | 比較明治維新的統一領導 |
| 外患加速崩潰 | 黃海一役見識瞬間歸零 | 甲午對洋務的致命衝擊 |

**步驟二（20 min）：SBQ 史料對讀**

教師提供 1–2 份原始史料（如李鴻章〈籌議海防摺〉節錄、嚴復〈原強〉節錄），學生將遊戲所學與史料互證：
- 「遊戲裡看到的，史料怎麼說？」
- 「史料的角度是誰？有什麼局限？」

**步驟三（10 min）：快速論文提綱**

限時寫一份 DSE 論文提綱（僅論點＋史例，不必成文）：
> 題目：「洋務運動的失敗，主要由於領導人的決策失誤。」你是否同意？

## 四、教師 Debrief 必做清單

每次使用後，以下問題**至少問其中兩條**：

1. 「你覺得自己的選擇改變了歷史嗎？——為什麼最後都走向甲午？」（反事實澄清）
2. 「遊戲的哪個部分，讓你覺得最貼近真實歷史？最不真實？」（史料批判）
3. 「如果你是洋務派的一員，你最大的困難是什麼？」（歷史神入）
4. 「器物、制度、思想——洋務做到了哪一層？為什麼止步？」（概念鞏固）
5. 「洋務派和明治維新的領袖，面對的問題一樣嗎？」（比較史觀）

## 五、差異化教學建議

| 學生類別 | 建議操作 |
|---|---|
| 學習支援生 | 開啟 Coach 引導（新手模式）；以「器物」路線為主，避免複雜懸案 |
| 一般生 | 自由選路線；完成至少一幕結局；填學習單 |
| 資優生 | 要求完成兩條路線，比較史評差異；嘗試懸案抉擇；課後延伸讀嚴復〈天演論〉 |

- 結算頁截圖作**學習歷程檔案證據**
- 以遊戲史評為素材，完成一篇 DSE 格式論文（300–500 字）

## 七、注意事項（教師必讀）

1. **遊戲≠教科書**：遊戲是動機工具與思維訓練，史實準確性仍以課本／史料為準。玩完後必須 debrief 補全時序與細節。
2. **反事實迷思**：條件選項會令學生誤以為「歷史可以改寫」。教師必須明確指出：七幕結局設計反映洋務的結構性困局，非個人能力問題。
3. **網絡要求**：遊戲為本地 Vite 項目，需教師電腦預先執行 `npm run dev`，或部署至校內伺服器後再讓學生使用。
4. **版權與使用**：本遊戲為教育用途自製項目，圖片資產均為合法使用，請勿商業轉發。
