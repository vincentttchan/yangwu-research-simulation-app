## 1. 已建成的系統（全部可運作）

### 流程主線
序章(開卷) → 擇路(輪播·4 路線·稟賦雷達) → 啟程 → D 過場(4 頁歷史 + 史料引語) → 領命(目標卡) → 入職過場 → 九州地圖

### 核心機制
- **年度回合制（方向乙）**：每年 1 個主動行動（探訪／研習各耗時）；跨城移動以 `cityTravelCost` 計季。
- **鐵釘事件**：`news`（傳信即觸發，不必親往）／`travel`（須親臨該城；設待辦 `pendingPinnedId` 鎖其他行動，抵城即觸發；`findDueTravelPinned` 處理趕路跳年/讀檔逾期）。
- **三軸見識**（器物/制度/思想）＝學到什麼；由求知考驗（DSE 題）與探訪/研習累積。
- **四方阻力資源**（印信面板「時局」）：聖眷/清議/民情/餉源，0–100，起始 55/50/50/45。年年欠餉；任一 <22 → **反撲**（軟懲罰+DSE 解說，每資源 4 年冷卻，`SETBACKS`/`checkResourceSetbacks`）。
- **effects 混合模型**：事件層 `ev.effects` + 選擇層 `choice.effects`。
- **★ Core Loop / 輕量任務（查證系統）**：點熱點 → 完成小任務（4 型）→ 收證據（+1 見識、解鎖探訪）→ 再點熱點「探訪 →」進事件。**詳見 `docs/城市CoreLoop-設計與實作-v2.md`**（設計/schema/流程/各城/擴充步驟）。
- **★ 丙·動態小事件（INTERLUDES）**：在城內推進落於「空檔年」（無鐵釘、無待辦）時，**必出一張不耗行動的時局插曲**（阻力示警/友訊/見聞，含 DSE 句）。`maybeShowInterlude`／`pickInterlude`／`#interludeModal`。
- **★ 史事主圖（事件圖）**：事件 modal 自動讀 `assets/events/<eventId>.webp`（`setEventImage`，無圖則優雅退場）。**16 張已生成並接線**（含已建事件＋終局 e_beiyang_fleet/e_yellow_sea_battle/e_shimonoseki_treaty 三圖**已備但事件未建**）。風格規格見 `docs/史事圖片資產清單與Prompt-v1.md`。
- **證據卡 reveal**：收錄→證據卡（朱砂「證」印＋「已入手卷」＋證據句橫排＋DSE/見識）→「探訪 →」一氣呵成（`showEvidenceReveal`）。註：曾試過直書拓片版、因橫直混排難讀已棄用。
- **書信引線**：事件後來信串連下一城 + 寄信人加入「友」（`LETTERS`/`deliverLetter`/`addToNetwork`）。
- **指引系統**：頂部「當前要務」橫幅。**v57 修復**：以 `gameState.pendingPinnedId` 為權威（含逾期待辦），永遠顯示「速往〔城〕」，不再空白或凍結（`updateGuidanceHint`）。
- **★ 支柱二·多線引導板**：`renderSideContent()` 在要務下方列「另可探訪·城③」可點擊 chip（已解鎖＋有選修內容的城＋線索數，點擊經 `onCityClick` 直達）；`countCityClues()`；強制待辦時隱藏。
- **★ 支柱三·懸案（需求拉力）**：`SUSPENSES` **11 樁**帶限期時局（分佈 1863–1891，含甲午前十年三樁）；每軸分佈：餉源3／聖眷3／清議3／民情2。及時處理穩住一方阻力（resolveGain +2、耗一季），**逾期 2 年 → 該阻力 −6**（`checkSuspenseExpiry` 跨年結算）。引導板紅警 chip → `openSuspense` 複用 interlude modal 解算。state＝`gameState.suspenses`（id→'resolved'|'expired'）。
- **★ 見識解鎖式**：`INTERLUDES.insight` 三軸各一張深層卡，`eligible` 的 `axisGate{axis,min:6}` 達標才現；`pickInterlude` 首見高優先。
- **★ 途中人物（城市綁定）**：見聞卡可帶 `city`（`eligible` 限該城才出現）＋選項 `meet{key,name,relation}`，`resolveInterlude` 結識即 `addToNetwork`＋`renderNetwork`。9 城各一人物。名冊常駐右側直欄。
- **★ 選項資源預覽**：`effectChips(effects)` 共用小籤（綠＋/紅−），接事件選項＋時局選項。
- **設施一次性**：`facilityIsStudy(f)`（有 challenge/gain）→ key `fac:id`（全程一次，顯「已研習」、場景打 ✓）；客棧（無 gain）維持每季可歇。

- **進城 loader 動畫**：墨色腳印漸次浮現（`cl-steps`）＋微塵（`cl-dust`）＋城名書法淡入（`cl-ink-bleed`）。
- **★ 玩法深度（v99）**：①**路線起始稟賦** `ROUTE_PERK`（`resetGameState` 末套用）——李鴻章 器物+2/聖眷+5；容閎 思想+2/清議+5/聖眷−5；奕訢 制度+2/聖眷+8；自由 三軸各+1/四阻各+2。②**失敗結局·罷官** `checkDismissal`（聖眷≤0 → `triggerDismissal`：收彈窗→提示→`showSettlement`，`gameState.dismissed` 旗標）；`buildVerdict` 加罷官前言、`showSettlement` 標題「罷官還鄉」。其餘軸歸零不即時出局（餉源 0＝慢性崩潰本意）。`gameOver` 旗標防重觸；存讀檔已接。
