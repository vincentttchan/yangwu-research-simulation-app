# Opening Four-Page Transition Revision: Page 4 as Era Summons

## Purpose

本方案處理「開場四頁轉場」與「人物選擇頁」之間的重複問題。現時第四頁以李鴻章作人物起點，內容與其後人物選擇頁的李鴻章介紹相近，令玩家在進入人物頁時感到資訊重複。方案 A 的核心是：保留四頁轉場的沉浸價值，但把第四頁改為所有玩家共同經歷的「時代召喚」，讓人物頁才正式承擔「選擇歷史視角」的功能。

## Design Position

四頁轉場不是教學負擔，而是遊戲開場的電影化引子。前三頁依次建立戰敗、圓明園焚毀、器物差距，已能有效交代晚清自強的歷史壓力。第四頁應由「人物介紹」改為「時代召喚」，功能是把前三頁的危機收束成一個玩家必須回應的問題：在內憂外患之間，清廷如何求強？玩家將以不同人物／路線進入這個問題。

## Revised Four-Page Structure

1. **第二次鴉片戰爭**
   - 功能：交代戰爭壓力與「天朝秩序」的動搖。
   - 保留現有方向。

2. **圓明園焚毀**
   - 功能：把戰敗轉化為情緒與象徵層面的創傷。
   - 保留現有方向。

3. **木帆船對鐵甲艦**
   - 功能：指出器物差距，使「學習西方技術」成為迫切問題。
   - 保留現有方向。

4. **時代召喚**
   - 功能：不介紹任何單一人物，而是邀請玩家選擇一條自強之路。
   - 由「李鴻章人物起點」改為「共同歷史問題」。
   - 視覺上可以沿用人物頁前的暗色電影語氣，但不出現李鴻章肖像。

## Visual Direction

第四頁不應使用人物肖像，而應使用「時代壓力」的畫面語言：

- 可用暗色地圖、焚後宮苑、海防／炮艦剪影，或幾張前面場景的淡疊影像。
- 中央或偏左呈現大字標題「局勢已動」。
- 畫面可有朱砂印、煙霧、紙卷、地圖線等元素，但避免變成教學圖示。
- CTA 要短，寬度約 220–260px，與首頁主按鈕一致。

## Relationship With Character Selection Page

人物頁負責「個別視角」，不再重複開場歷史背景。以李鴻章為例，人物頁應聚焦：

- 他的起點：上海、江南製造、北洋伏線。
- 他的困局：船炮可造，但制度與朝局未必肯動。
- 玩家選擇他的意思：從器物自強出發，逐步看見制度、財政、朝議與民情的牽制。

因此，人物頁可以保留「李鴻章」與核心矛盾，但不再需要再次重述圓明園、戰敗、木帆船對鐵甲艦等背景。

## Interaction Rhythm

建議保留現有四頁轉場的沉浸節奏：

首頁「入局」 → 四頁開場轉場 → 人物選擇頁 → 「擇此人物」進入選定狀態 → 第二次「啟程入局」作為儀式性確認 → 短轉場 → 地圖

第二次「啟程入局」不應刪除。它應被設計成玩家正式承諾進入某條歷史路線的儀式，而不是普通確認按鈕。

## Acceptance Criteria

- 第四頁不再顯示李鴻章肖像、姓名、引文或個人起點。
- 第四頁清楚把前三頁危機收束為「選擇自強之路」。
- 人物選擇頁第一次正式介紹李鴻章等人物。
- 首頁、第四頁、人物頁的 CTA 寬度與語氣統一，不再出現過長紅色框。
- 過場仍保留四頁，不縮短為純跳轉。
- 不改動研究 logger、後台數據或版本記錄範圍。

## Open Points For Review

1. 第四頁標題是否使用「局勢已動」，還是改成更有史詩感的「求強之路」或「自強之局」？
2. 第四頁 CTA 是否用「擇路入局 →」，還是保留較有氣勢的「入局啟程 →」但縮短視覺寬度？
3. 第四頁視覺是否應偏向地圖、焚毀宮苑、還是海防／船炮剪影？
