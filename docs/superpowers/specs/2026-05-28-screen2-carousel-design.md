# Screen 2 全幅旋轉木馬 · 設計文件

**日期**：2026-05-28  
**專案**：自強三十年 · 探索版  
**範疇**：將 Screen 2「擇路」的 2×2 格局完全替換為全幅旋轉木馬

---

## 背景

現有 Screen 2 以 2×2 格局並排顯示四名人物路線。每格空間不足，銅版畫肖像難以突出，人物背景與遊玩資訊無法充分展示。

新方案改為每次只顯示一名人物，佔滿整個視窗，讓玩家在儀式感中逐一認識每條路線。

---

## 設計決定摘要

| 面向 | 決定 |
|------|------|
| 整體佈局 | 全幅旋轉木馬（B方案） |
| 肖像呈現 | 滿幅底圖・文字疊加（A方案） |
| 切換動畫 | 淡入淡出 crossfade（Y方案） |
| 選擇流程 | 已擇狀態 + 底部啟程欄（R方案） |

## 每一幀的結構

```
┌──────────────────────────────────────────────────────┐
│  文字層（左 40%）           │  肖像（右 60%，撐滿幀）  │
│                             │                          │
│  壹                         │   [銅版畫肖像 PNG]        │
│  李 鴻 章                   │                          │
│  Li Hongzhang · 1823–1901   │   右側撐滿，向左以        │
│  ──── (朱砂線)              │   linear-gradient 漸隱    │
│  [傳記文字，約80字]          │   融入暗色背景            │
│  器物派  外交重臣            │                          │
│  ★★★★★ · 約35分鐘 · 4城市  │                          │
│                             │                          │
│  [擇此路線 →]               │                          │
│                             │                          │
│  大字背景：壹（opacity 4%）  │                          │
└──────────────────────────────────────────────────────┘
‹ 左箭嘴              ● ○ ○ ○ (dots)           右箭嘴 ›
```

### 文字層元素（由上至下）

1. **漢字序號**：壹/貳/參/肆，小字，朱砂色，opacity 60%
2. **人物名**：大楷，letter-spacing 0.2em，`#e8d5a3`
3. **英文名 + 年份**：細小，`#7a6a4a`
4. **朱砂橫線**：18px 寬，`#cc3333`
5. **傳記文字**：約80字，`#a08a5a`，line-height 1.7
6. **標籤群**：派系 + 難度星，小膠囊樣式
7. **元資訊**：遊玩時間 · 鐵釘事件數 · 城市數，極小字
8. **CTA 按鈕**：`擇此路線 →`，朱砂邊框，選後變 `✦ 已擇定`

### 肖像層

- 絕對定位，`right:0; top:0; width:60%; height:100%`
- 圖片 `object-fit: cover; object-position: center top`
- 疊加 gradient：`linear-gradient(90deg, rgba(8,5,3,0.95) 0%, rgba(8,5,3,0.15) 60%, transparent 100%)`
- 大漢字背景數字：`right:5%; bottom:-4%`，`font-size: clamp(5rem, 12vw, 9rem)`，`opacity: 0.04`

---

## 切換動畫

**淡入淡出（crossfade）**

```
transition: opacity 0.6s ease
```

- 當前幀：`opacity 1 → 0`
- 下一幀：`opacity 0 → 1`（同時開始，短暫交疊）
- 過渡中禁用箭嘴點擊（`pointer-events: none`，0.6s 後恢復）
- Dots 同步切換 active 狀態

## 選擇流程（R方案）

### 狀態機

```
初始：全部幀未選定
  ↓ 點擊「擇此路線 →」
已擇：
  - 當前幀 CTA 變為「✦ 已擇定」（朱砂填色）
  - 底部確認欄滑出（translateY: 100% → 0，0.3s ease-out）
  ↓ 繼續撥動至其他人物
選擇轉移：
  - 舊幀 CTA 恢復「擇此路線 →」
  - 新幀 CTA 變「✦ 已擇定」
  - 確認欄內人物名更新
  ↓ 點擊「啟程 →」
進入 Screen 3：
  - Screen 2 整體淡出
  - 呼叫現有 startGame(routeKey) 函數
```

### 底部確認欄結構

```html
<div id="s2-confirm-bar">
  <span class="cb-name">李鴻章 · 洋務三十年</span>
  <button class="cb-start">啟 程 →</button>
</div>
```

- 初始 `transform: translateY(100%)`，選定後 `translateY(0)`
- 高度約 52px，背景 `rgba(8,5,3,0.95)`，頂部一條朱砂線
- 人物名居左，啟程按鈕居右

## 自由書記的特殊處理

| 元素 | 內容 |
|------|------|
| 序號 | 肆 |
| 人物名 | 自 由 書 記 |
| 英文 | Custom Route |
| 肖像 | 空白銅版畫框，刻有「？」符號 |
| 傳記 | 「不依附任何大人，自由穿行各城之間，記錄所見所聞。歷史的走向，由你選擇觀察的角度。」 |
| 標籤 | 自由探索（無難度星，無事件數） |
| 元資訊 | 時長不定 · 城市不限 |

## 導覽控制

## 響應式行為

| 寬度 | 處理 |
|------|------|
| > 1024px | 完整佈局，文字層 40%，肖像 60% |
| 768–1024px | 文字層 50%，肖像 50% |
| < 768px | 肖像縮至上半幀（height: 55%），文字移至下半幀 |
| < 480px | 箭嘴尺寸加大至 48px，dot 加大至 8px |
