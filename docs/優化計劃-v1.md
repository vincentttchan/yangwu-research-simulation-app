# 《自強三十年》優化計劃 v1

> 來源：四種身份體檢（機制／視覺／UX／玩家）＋ 三系統深化 ＋ 熱點校對 ＋ 背景地圖重製。
> 本文件為單一真相；完成項以 ✅ 標記。

## 已鎖定的設計方向（拍板）

| 主題 | 決定 |
|---|---|
| **見識（器物/制度/思想三軸）** | **A 解鎖式**：某軸達門檻 → 開放專屬事件／更深層解讀。見識＝「眼界」，眼界高才看得見更深的歷史。 |
| **時局（空年插曲 INTERLUDES）** | **內容擴充**：每個阻力池 3–4 張、見聞 8–10 張，按年代分層；每張掛 DSE 句。 |
| **舊友新知（網絡）** | **A 人物專屬插曲**：結識某人後其專屬時局卡進池（容閎談留學／徐壽談譯書／王韜談變法），人物成為「持續的聲音」。 |
| **熱點校對** | 類型 **(a)**：城市場景圖內的熱點**圓點**沒落在對應地物上 → 全城逐一截圖對位、批次修座標。 |
| **背景地圖** | Colton 1855《Asia, Eastern Sheet》裁切（107–143°E / 18–43°N，含朝鮮＋日本＋武漢、捨內亞）→ **img2img 保真風格轉換**（去飽和、虛化雜註、去花邊）。 |

## 三系統「根病」

三者都只「收集＋展示」，不回饋進玩法：見識只在結算顯示、時局一張卡見底、舊友結識後沉睡。
優化主軸＝**讓三者都回饋進「時局」這個容器**：人物插曲與見識門檻都灌進同一套 modal，三邊受益。

### Phase 2.5 — 節奏結構解（空檔季「沒有一季是死的」）
> 根因＝三個結構性錯配：**顆粒度**（迴圈單位是「季」120 個，內容按「事件」寫約 40–50 個）、**引導**（單線，只指下一鐵釘，藏起選修內容）、**動機**（選修探訪對生存無關，無拉力）。
> 拒用方案：磨見識（可溫故）、提早解鎖——只補第一層，不治本。
> 推進次序：**引導二 → 動機三 → 顆粒一**。
