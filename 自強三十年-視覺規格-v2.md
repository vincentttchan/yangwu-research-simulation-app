## 0. v2 與 v1 的關鍵差異

| 項目 | v1（廢棄） | v2（現行） |

| 設計比例 | Horizonte 50% / 中式 50% | **Horizonte 80% / 中式微妙暗示 20%** |

## 1. 設計哲學 v2

### 1.1 四個關鍵詞
| 詞 | 含義 | 表現手法 |
|---|---|---|
| **氣聲（Whisper）** | 一切文字都比預期更淡 | 對比度 1.5–3:1 而非常見 7:1 |
| **儀器（Instrument）** | 互動元素帶刻度／同心圓 | 按鈕 + 細圓 outline、圖釘 + ping |
| **景深（Depth）** | 至少 3 層視覺空間 | 霧化底圖 / 中層線描 / 前層文字 |
| **編輯（Editorial）** | 不對稱、巨數迷你標、留白主導 | 數字 80px + 標籤 9px、極簡網格 |

### 1.2 中式肌理的注入位置（只此三處）
1. **底層素材的氣質**：清末玻璃版攝影、文人山水墨色（霧化處理後當底圖）
2. **字體的東方骨架**：思源宋體 300/400 字重，**永不**用 900
3. **點睛朱砂的時機**：朱砂只在鐵釘事件首次觸發、現在所在城市時出現

**禁用**：朱印方框、雲紋頁腳、紙紋疊加（v1 全部移除）、彩色木版年畫風格

## 3. 字體系統 v2

### 3.1 字族
| 用途 | 字體 |
|---|---|
| 中文襯線 | Noto Serif TC（思源宋體繁中），權重 300/400/500 |
| 西文襯線 | **Cormorant Garamond Italic** ← 關鍵：必須是 italic 而非 light |
| 西文等寬 | JetBrains Mono 400 |
| 數字 | Cormorant Garamond 300（細） |

### 3.2 「斜體節奏」中文解法（v2 修正）
v1 用「中文 700 vs 300」對比 — **失敗**，中文沒有真正的義體。

v2 解法：**西文 italic 短語穿插中文**。範例：

```
踏 遍 九 州  across nine provinces  的 書 記 之 旅
```

- 中文用 Noto Serif TC 400
- 西文 italic 用 Cormorant Garamond Italic
- 西文字級為中文的 60%、顏色 `--mist`
- 兩者之間留 0.6em 空隙

這還原了 Horizonte 「LOCATION *in* RELATION」的真實雙聲部，**而非用字重硬撐**。

### 3.3 字級表
| 角色 | 桌面 | 平板 | 手機 |
|---|---|---|---|
| 引言巨標 | 64px / 400 / 1.25 | 52px | 40px |
| 章節標 | 40px / 400 | 32px | 26px |
| 西文 italic 註腳 | 14px / italic | 13px | 12px |
| 數字（巨） | Cormorant 80px / 300 | 64px | 48px |
| 數字標籤 | 9px / mono / 全大寫 / 字距 0.25em | 同 | 同 |
| 說明文 | 15px / 1.7 | 14px | 13px |

## 4. 動態語言（核心章節）

### 4.1 三級音量

| 等級 | 用法 | 頻率 |
|---|---|---|
| **fortissimo 強** | 一局只 1–2 次 | 開頁載入、首次海岸線描繪、朱砂第一次出現 |
| **mezzo 中** | 章節事件 | 鐵釘事件淡入、季節轉換、移動方框蓋落 |
| **pianissimo 弱** | 持續環境 | 同心圓呼吸、音軌浮動、文字微反饋、自訂游標 |

**最忌諱**：中音量持續（到處都在動），會疲勞。

### 4.2 通用動畫清單

#### ① 載入序列（fortissimo）
- 全屏 `--parchment` 背景
- 中央顯示巨大數字「1860」，Cormorant 300，160px
- 在 2.8 秒內**跳動到 1895**，使用 `easeOutCubic`
- 下方淡入「自強三十年 · 踏遍九州的書記之旅」
- 完成後年份淡出、主畫面內容 stagger 進場

#### ② Smooth Scroll（pianissimo，桌面才用）
- 自寫 lerp 滾動（30 行 JS）：`current += (target - current) * 0.08`
- 手機保留原生滾動

#### ③ 文字 reveal（fortissimo / mezzo）
- 引言頁主標題**逐行**從下往上 8px 滑入 + 淡入
- stagger 0.15s
- 每行 0.8s ease-out

#### ④ 自訂游標（pianissimo）
- 一個 6px 點 + 一個 36px 淡圓外圈
- 點即時跟隨、外圈 lerp（lag 0.12s）
- hover 在可點擊物：點縮為 3px、外圈擴為 48px
- **設置可關閉**（`prefers-reduced-motion` 自動關）

#### ⑤ Magnetic Button（mezzo）
- 只用於「開卷」與鐵釘事件「閱卷」
- 游標距離按鈕 ≤120px 時，按鈕向游標位移最多 6px
- 離開後 spring back

#### ⑥ 同心圓呼吸（pianissimo）
- PLAY/開卷按鈕外圍 2 圈 1px 細圓，半徑 110% 與 130%
- 各自做 `scale(1) → scale(1.15) + opacity(0.4 → 0)` 循環
- 週期 3.6s，兩圈相位差 1.8s
- hover 時加速到 1.8s 週期

#### ⑦ 景深層（mezzo，非滾動視差）
- 底層：霧化攝影／繪畫 8–15% 不透明 + blur 0.5px
- 中層：線描素材 25% 不透明
- 前層：文字、UI（100%）
- 鼠標移動時，中層**極輕**地反向位移 ±3px（製造景深暗示）

#### ⑧ 音軌呼吸（pianissimo）
- 12 條豎條，各自 height 動畫
- 用 CSS keyframes、每條延遲不同
- 整體音量很低（看得到、不搶戲）

### 4.3 兩個獨佔的歷史敘事動畫

#### A. 朱砂墨滲（fortissimo，一局僅一次）
- 1860 圓明園鐵釘事件觸發時，**朱砂首次出現**
- 從中心點向外**不規則擴散**（用 SVG `feTurbulence` 變形邊緣）
- 1.2 秒緩慢渲染
- 配合一聲低沉「印章蓋落」音（程序生成 100Hz + 60Hz 衰減）

#### B. 海岸線逐筆描繪（fortissimo，僅首次進地圖）
- 首次進入地圖：中國海岸線從北方（黑龍江）逐筆描出到南方（海南）
- SVG `stroke-dasharray` + `stroke-dashoffset` 動畫
- 3 秒，ease-in-out
- 完成後 10 個城市標記 stagger 0.08s 浮現
- 最後「現在所在」的同心圓開始呼吸
- localStorage 記錄已播放，二次進入不重播

## 5. 素材策略

### 5.1 底層素材
**首選**：1860–1900 公版攝影（John Thomson、Felice Beato 中國照片）
**次選**：清末文人山水（公版）
**備案**：用 AI 圖工具（Midjourney / DALL-E）將公版照片**風格化統一**為單一色調

### 5.2 處理規範（所有底層素材）
1. 去飽和到單色（沿 `--ink-soft` 偏暗的色階）
2. 不透明度降到 8–15%
3. 高斯模糊 0.5–1px
4. 暖色平衡向 `--parchment` 偏移
5. **首選用 webp 格式，每張 < 80KB**

### 5.3 中層線描素材
- 清末教科書、申報插圖、點石齋畫報（皆公版）
- 主題：圓明園殘柱、江南製造局、北洋艦影、總理衙門
- 處理：單色 `--ink-soft`、25% 不透明
- **若公版難尋，可用 AI 統一風格化**

### 5.4 禁用
- AI 生成的地圖（地名可能錯）
- AI 生成的歷史人物肖像
- 任何彩色插畫
- v1 用過的紙紋疊加、雲紋頁腳

## 8. 工程實作 v2

### 8.1 檔案結構
```
探索版/
├── index.html
├── style-explore.css
├── explore.js          # 主邏輯
├── motion.js           # 動畫工具（lerp scroll, cursor, magnetic）
├── intro.js            # 引言頁專用
└── assets/
    ├── sketches/       # 線描素材
    │   ├── ruins-yuanmingyuan.svg
    │   └── ...
    └── map/
        ├── basemap.webp        # 霧化底圖
        └── coastline-trace.svg # 動畫用海岸線路徑
```

### 8.2 不引入的依賴
- ❌ Lenis（自寫 30 行 lerp scroll 即可）
- ❌ GSAP（用 CSS keyframes + Web Animations API）
- ❌ 任何字體載入庫（用原生 `font-display: swap`）

### 8.3 性能預算
- 首屏 LCP < 2.5s
- JS 總和 < 50KB gzipped（比 v1 預估的 80KB 更緊）
- 字體子集化（中文約 800 字）
- 底圖 webp < 80KB

### 8.4 無障礙
- `prefers-reduced-motion`：關閉所有動畫、隱藏自訂游標
- 所有互動有鍵盤 fallback
- 對比度雖低，但**互動文字**（按鈕、連結）必須 ≥4.5:1（用 `--ink-soft` 或 `--ink-deep`）
